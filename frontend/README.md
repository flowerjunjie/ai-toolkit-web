# AI Toolkit Web Interface

**Web界面让非技术用户也能轻松使用AI Toolkit的2052+命令！** 🚀

## 🎯 项目目标

为AI Toolkit创建友好的Web界面，让：
- 💼 商务人士也能用AI工具
- 📊 研究人员快速分析数据
- 🎨 创作者轻松生成内容
- 🎓 学生/教育者简化操作

## 📐 技术栈

### 前端
- ✅ React 18 + TypeScript
- ✅ Ant Design 5.x (企业级UI)
- ✅ Zustand (状态管理)
- ✅ React Router 6
- ✅ Vite (构建工具)
- ✅ TailwindCSS

### 后端 (即将开发)
- ⏳ FastAPI (Python)
- ⏳ RESTful API
- ⏳ JWT认证

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
